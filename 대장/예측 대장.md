# 예측 검증 대장

SemiAnalysis 문서에서 추출한 **시점이 박힌 예측**을 축적하고, 시간이 지나면 실제 결과와 대조해 검증 상태를 갱신하는 대장입니다.

## 사용 안내

- **추가 시점**: TRANSFORMATION_RULES.md v17.0 규칙에 따라 문서 변환 완료 직후 append (문서 단위 섹션)
- **섹션 형식**: `## [YYMMDD] 문서 제목` 아래에 표 하나
- **표 컬럼**:
  | 예측 내용 | 주체 | 목표 시점 | 근거 요약 | 검증 상태 |
  - **주체**: `저자`(SemiAnalysis 자체 분석) / `업체`(기업 발표·로드맵 인용) / `기관`(정부·리서치 기관)
  - **검증 상태**: ⏳ 시점 미도래 / ✅ 적중 / 🔶 부분 적중 / ❌ 빗나감 — 갱신 시 근거 각주를 함께 남김
- **검증 갱신**: 새 문서 변환 중 과거 예측과 관련된 사실을 발견하면 해당 행의 검증 상태를 갱신하고, 출처(새 문서명)를 병기
