# 통합 인사이트 — 언더스탠딩 대시보드 크로스-문서 합성

작성일: 2026-07-25
대상: `대시보드/언더스탠딩 대시보드.html`

## 1. 목적 / 문제

언더스탠딩 대시보드는 23개 카드(백브리핑 13 + 권효재 9 + 이선엽 1)를 6개 산업 섹션으로 나눠 보여준다. 하지만 사용자는 **개별 카드 나열이 아니라, 관련 문서들이 합쳐서 말하는 통합 인사이트**를 원한다. 섹션 구분과 무관하게, 주제로 묶인 문서군에서 "연결된 결론"을 뽑아 상단에 정적으로 제시한다.

## 2. 요구사항 (확정)

- **출력**: 정적으로 미리 써둔 합성 문서(읽기용). 런타임 LLM 호출 없음.
- **묶음**: 저자가 23편 전체에서 자동 도출한 주제 클러스터. 클러스터당 통합 인사이트 1블록.
- **범위**: 백브리핑 + 권효재 + 이선엽 전체(에너지·반도체 겹침 포함).
- **배치**: 언더스탠딩 대시보드 상단 새 "통합 인사이트" 섹션(기존 6섹션 위). 각 블록이 근거 카드로 링크.
- **구성(블록)**: `통합 논지` → `공통 진단` → `연결·인과` → `상충·이견` → `한국 함의` → `근거 카드` 링크.
- **비목표(YAGNI)**: 동적 생성, 질문형 RAG, 자동 클러스터링 파이프라인, ouroboros 연동.

## 4. 데이터 모델

클러스터당 md 1개: `content/understanding/통합/<ID-슬러그>.md`

```yaml
---
cluster_id: A
title: "이란 전쟁과 원유 수급 충격"
subtitle: "해협 봉쇄가 유가·LNG·정유로 번지는 사슬"
sources:              # 대시보드 카드 <h2> 제목과 정확히 매칭
  - "이란 전쟁 오늘 끝나도 유가 9월까지 뛴다 (1부)"
  - "호르무즈 해협 봉쇄 맞불… 하르그섬 뭐길래?"
updated: 2026-07-25
---
```
본문(구조화):
```markdown
## 통합 논지
## 공통 진단
## 연결·인과
## 상충·이견
## 한국 함의
```

## 5. 컴포넌트 / 데이터 흐름

- **소스(진실원천)**: `content/understanding/통합/*.md` (합성 원문, 버전관리).
- **빌드**: `scratchpad/gen_insights.py`
  1. `통합/*.md` 파싱(프론트매터 + 본문).
  2. 대시보드 각 카드 `<h2>`에 `id="card-<슬러그>"` 부여(멱등).
  3. 본문 md→HTML: 제한 서브셋(`##` 소제목, `-` 불릿, `**bold**`, 문단) 자체 렌더러. 외부 의존 없음.
  4. `sources` 제목 → 카드 앵커(`#card-…`) 링크. 미매칭 제목 경고.
  5. "통합 인사이트" 섹션 HTML을 nav 위에 주입. `<!-- INSIGHTS:START -->`…`<!-- INSIGHTS:END -->` 센티넬로 멱등 재생성. nav 맨 앞에 칩 `🧭 통합인사이트 N` 추가.
- **표시**: 대시보드 상단 통합 섹션 + 6개 산업 섹션(기존).
- **실행 순서**: `regroup_dashboard.py`(6섹션·정렬) → `gen_insights.py`(카드 id + 통합 섹션). gen_insights를 마지막에.

## 6. 에러 처리 / 엣지

- `sources` 미매칭 → 경고 + 해당 링크 스킵(무크래시), 미매칭 목록 출력.
- 카드 슬러그 충돌 → 접미 번호로 유일화.
- `통합/` 없음 / md 0개 → 통합 섹션 스킵, 정상 종료.
- md→HTML 서브셋 외 문법 → 평문 통과.

## 7. 검증 (빌드 후 자동)

- `<div>` open/close 균형.
- `<!-- INSIGHTS:START/END -->` 존재, 클러스터 블록 4개.
- nav 칩 수 = 통합 1 + 산업 6 = 7.
- 모든 `sources` 매칭(경고 0).
- 모든 `#card-…` 앵커에 대응 `id` 존재.
- (선택) 아티팩트 리로드 스크린샷 — 브라우저 응답 이슈 시 스킵.

## 8. 갱신 흐름

새 문서 추가 → 카드 생성(기존 파이프라인) → 해당 `통합/*.md`의 `sources`·본문 갱신 → `regroup` → `gen_insights` 재실행 → 아티팩트 재발행 + 커밋/푸시.

## 9. 저작권 주의

통합 인사이트는 저자(합성)의 창작이며 유료 원문 전문을 재수록하지 않는다. 근거 링크는 기존 카드(요약)와 유료 원본(외부)만 가리킨다.
