## 🔑 용어 정리

본문을 순서대로 읽기 전에 알아두면 좋은 용어들입니다. 자세한 수치와 설명은 본문에서 처음 등장하는 위치에 나옵니다.

- **ARR (Annual Recurring Revenue, 연환산매출)**: 지금 이 달의 매출 속도를 1년치로 환산한 값 — 실제 지난 1년 실적이 아니라 "지금 속도가 유지되면 연간 얼마"라는 스냅샷
- **NNARR (Net New ARR, 순증 연환산매출)**: 한 달 동안 새로 늘어난 ARR — 신규 고객 유치분과 기존 고객 사용량 증가분을 합친 월간 성장 속도
- **TaaS (Token-as-a-Service)**: 고객이 모델 개발사(랩)에서 직접 사지 않고 AWS·구글 같은 클라우드를 거쳐 토큰(모델 사용량)을 구매하는 간접 유통 방식
- **EBTIT (Earnings Before Training, Interest, and Taxes)**: 신규 모델 학습 비용을 빼기 전 이익 — AI 랩이 지금 얼마나 재투자할 실탄을 쥐고 있는지 보여주는 지표
- **GAAP EBIT (일반회계기준 영업이익)**: 미국 회계기준(GAAP)으로 계산한 영업이익 — 스톡옵션 등 비현금성 비용까지 반영해 기업이 자체 발표하는 비GAAP 지표보다 보수적
- **Net Dollar Retention (순매출유지율)**: 1년 전부터 있던 기존 고객들의 매출이 지금 몇 %로 늘거나 줄었는지 나타내는 비율 — 신규 고객 없이 기존 고객만으로 매출이 얼마나 느는지 측정
- **ARR per MW (MW당 매출)**: 컴퓨트(전력) 1메가와트를 굴려서 만들어내는 연환산매출 — 같은 하드웨어로 얼마나 돈을 잘 버는지 나타내는 효율 지표

## 1. 서론: 앤트로픽 IPO가 던지는 질문

**📌 핵심:**
- 앤트로픽은 2026년 6월 1일 비공개로 IPO를 신청 — 앤트로픽·OpenAI 합산 ARR(연환산매출)이 이미 약 1,000억 달러에 달하는 가운데, 두 회사 중 앤트로픽이 먼저 움직임
- 3분기(3Q26) 기준 앤트로픽은 GAAP 영업이익(EBIT) +10억 달러(매출 대비 6% 마진)로 흑자인 반면, OpenAI는 영업이익률 -100%로 여전히 대규모 적자 — 이 격차가 두 회사의 자금조달 우위를 가름
- OpenAI는 자체 IPO를 2027년으로 미룬다는 보도가 나오면서, 두 회사 모두 상장을 통해 자금을 조달할 능력이 있는지에 대한 의문이 제기됨
- 결론: 비공개 신청이라 공식 수치는 없지만, SemiAnalysis Tokenomics팀이 SKU·요금제·고객 유형별로 상향식(바텀업)으로 추정해온 재무모델을 최근 WSJ 보도가 정확성 면에서 뒷받침 — 이 문서는 그 추정 재무제표를 공개

## 2. IPO 배경 - 왜 지금 상장하는가

**📌 핵심:**
- 앤트로픽은 이미 1,000억 달러 넘게 조달했고 비GAAP 기준으로는 흑자인데도 상장을 택한 이유는 3가지 — ① 주식·채권 시장 접근(향후 필수) ② 감사된 공개 재무제표로 대형 계약 체결이 쉬워짐 ③ 직원의 지분 매각을 쉽게 해 인재 전쟁에서 우위
- 2030년까지 OpenAI·앤트로픽 합산 컴퓨트 수요가 100GW를 넘을 전망 — 2025년에 겨우 2.5GW, 2026년 5GW가 추가된 것과 비교하면 2030년까지 90GW 이상의 순증설이 필요한데, 현재 두 회사가 확보한 컴퓨트는 약 6GW뿐
- 1분기(1Q26) 동안 사용자들이 이 컴퓨트 부족을 체감 — 사용 제한(rate limit), 다운타임 등으로 Claude 이용이 어려워짐. 앤트로픽 API 사업의 매출총이익률이 이미 매우 높아, 공개시장 자금 조달로 이 마진을 레버리지 삼아 경쟁사가 따라오기 힘든 수준으로 컴퓨트 확보 경쟁에서 우위를 점할 수 있음
- 결론: 구글이 6월 848억 7,500만 달러를 증자하고 메타도 수백억 달러대 조달설이 도는 등 하이퍼스케일러조차 주식 발행에 나선 것은 AI 컴퓨트 수요가 그만큼 강력하다는 신호 — 앤트로픽 IPO는 이 자금조달 경쟁에 참여하려는 시도

## 3. 사업 구조 - 구독과 API 매출 믹스

**📌 핵심:**
- ChatGPT가 소비자에게 "아이폰의 순간"이었다면, Claude Code는 B2B(기업간거래)에서의 변곡점 — Claude Code는 이제 전체 GitHub 커밋의 7% 이상을 차지하며 2026년 초 앤트로픽 ARR 급가속을 이끔(2025년 말 90억 달러 → 현재 600억 달러 이상)
- 앤트로픽 매출의 75\~85%는 사용량 기반 API 매출, 구독은 15%에 불과 — 반대로 OpenAI는 역사적으로 소비자·구독 중심(1Q26 기준 65%+가 구독)이었으나 최근 변화 중
- 소비자 구독은 Free·Pro($20/월)·Max 5x($100/월)·Max 20x($200/월) 4단계, 팀(Team) 구독은 5\~150석 기업 대상 $20 또는 $100/월(연간 결제 시), 엔터프라이즈 구독은 별도 사용량 요금 없이 150석 초과 시 API 요금이 그대로 부과됨
- 결론: API 매출은 좌석(seat)이 아니라 토큰 단위로 과금되어 매출 상한이 없음 — 같은 고객이 에이전틱 워크플로우를 더 많이 쓸수록 신규 고객 없이도 매출이 계속 늘어나는 구조

📌 용어 풀이: 왜 "ChatGPT=소비자 아이폰 모먼트, Claude Code=B2B 변곡점"인가
> - ChatGPT는 일반 소비자가 처음으로 AI를 일상적으로 쓰기 시작한 계기였다면, Claude Code는 기업(개발팀)이 AI에 실제 업무를 맡기기 시작한 계기라는 의미
> - 이 차이가 두 회사의 매출 구조(소비자 구독 vs 기업 사용량 과금)를 갈랐다는 것이 저자들의 핵심 논지

### 매출 구조 - API 75\~85% vs 구독 15%

반면 OpenAI는 역사적으로 소비자·구독 중심 사업(1Q26 기준 구독 비중 65%+)이었으나, 최근 이 구도가 바뀌는 중입니다. 소비자 구독 비중만 봐도 격차가 뚜렷합니다 — 앤트로픽 ARR 중 소비자 구독은 단 5%인 반면, OpenAI는 2Q26 말 기준 약 40%로 추정됩니다(OpenAI 최고고객책임자도 1Q26 블로그에서 B2B가 전체 ARR의 약 40%라고 언급).

결국 랩(모델 개발사)의 사업 모델은 단순합니다 — 신규 모델을 만들고, 이를 학습·서빙할 컴퓨트를 사거나 빌린 뒤, 그 컴퓨트를 토큰 단위로 고객에게 판매하는 구조입니다. 앤트로픽과 OpenAI는 이 컴퓨트를 판매하는 방식이 역사적으로 크게 달랐습니다 — 앤트로픽은 고마진 사용량 기반 가격 정책에 사업 모델 전체를 맞춘 반면, OpenAI는 비용이 많이 드는 무료·구독 사용자를 다수 서빙하며 컴퓨트를 대량으로 소모해왔습니다. OpenAI가 점차 앤트로픽 방식으로 수렴하며 신규 ARR이 B2B·API 위주로 바뀌고 있지만, 여전히 소비자·구독 기반 사용량이 재무제표에 상당한 부담을 주고 있습니다.
